# Starter

Template for practicing basic HTML, CSS, and Javascript syntax.

## 如何開始？

```shell
# 下載程式碼
git clone https://github.com/Chang00Z/starter.git

# 移動至程式所在 folder
cd starter

# 安裝相關套件
npm install

# 在本機上運行
npm run dev
```

運行成功後，會看 terminal 中看到以下訊息，於瀏覽器開啟該連結，即可看到練習畫面

```
➜  Local:   http://localhost:5173/
```
